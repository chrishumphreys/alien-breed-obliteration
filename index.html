Alien Breed Obliteration in Python for Wii

These pages describe my journey from idea to release of a Wii Homebrew game. The game in question is a remake of the classic Amiga Alien Breed games. This version is a clone of a PC retro remake Alien Breed Obliteration.

The game is now released and all source and binaries are available on the wiiab Sourceforge project page.
