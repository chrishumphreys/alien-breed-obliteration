Whilst the Wii version requires the custom Mygame library and must be frozen down to an ELF executable, the game itself can be run on other platforms using pure Python/Pygame. This makes it very portable.

The sourceforge project includes some portable python code to perform the equivalent of the Wii installer allowing a pure python distribution for other platforms.

The binary files for Linux/Windows/OS X are available here. Note you want the crossplatform pyab archive.

Update:
Looks like the pure Python version has subsequently been ported by others to AmigaOs4 and GP32 Pandora.

It has subsequently been ported to Java to run on android
